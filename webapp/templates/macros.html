About dbt Macros

Macros in dbt are like functions in other programming languages. They allow you to:

  • Reduce code duplication
  • Abstract complex SQL into reusable components
  • Implement conditional logic
  • Define custom schema tests

Macros are written using Jinja, a templating language for Python.

Package Macros

In addition to project macros, this dbt project uses macros from these packages:

  • dbt_utils - A collection of useful macros
  • dbt_date - Date manipulation utilities
  • dbt_artifacts - Track run history and metadata
