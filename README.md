# dbt_timeline_analysis
Sample webapp to visualize the execution timeline of a dbt project. Built with Replit as a POC and inspired by [Airbnb's "Visualizing Data Timeliness at Airbnb" blog post](https://medium.com/airbnb-engineering/visualizing-data-timeliness-at-airbnb-ee638fdf4710).
